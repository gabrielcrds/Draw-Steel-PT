## **Dados**

Esse jogo usa dados de dez lados (chamados de d10). Cada jogador, incluindo o Diretor deve ter dois desses. Alguns dados de dez lados são numerados de 0 a 9, enquanto outros são numerados de 1 a 10. No caso dos primeiros, o 0 conta como 10. O jogo também ocasionalmente usa dados de seis lados (chamados de d6), então é bom que cada jogador tenha um ou dois desses também.

**D3s**

Raramente, as regras pedem a um jogador lançar um ou mais dados de três lados (chamados de d3). Se você não tiver um d3, pode lançar um dado de seis lados e tratar um lançamento de 1–2 como 1, um lançamento de 3–4 como 2 e um lançamento de 5–6 como 3.

**D100s**

Algumas tabelas no jogo pedem um lançamento de d100. Pra lançar um d100, use dois dados de dez lados. Decida qual dado representa as dezenas, com o outro dado representando as unidades. Por exemplo, se lançar 5 nas dezenas e 3 nas unidades, o número lançado é 53.

Alguns dados de dez lados são numerados de 0 a 9, enquanto outros são numerados de 1 a 10. No caso desses últimos, o 10 conta como 0 para lançar um d100. Por exemplo, se lançar 10 nas dezenas e 9 nas unidades, o número lançado é 09, ou 9. Se ambos os dados lançados tirarem 0 ou 10, então o número lançado é 100!
